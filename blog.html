Question-1: What are the different ways to select an element in the DOM?

------------------------------------------------------------------------------------------------------------------

Ans: There are few ways to do that. Like:
a. By using id- getElementById
b. By class- getElementsByClass
c. By tag name- getElementsByTagName
d. By css query selector- querySelector
e. By css query selector all- querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

------------------------------------------------------------------------------------------------------------------

a. innerHTML is the HTML code inside of any selected element.
b. innerText is the string value inside any html tags or elements. It doesn't read any text that's hidden using css.
c. textContent is similar to innerText. Just it can read all the text related contents of an element whether it's hidden or shown.

Question-3: What is event delegation in the DOM?

------------------------------------------------------------------------------------------------------------------

The process of attaching an event listener with the parent element instead of different child elements is delegation of event in DOM

Question-4: What is event bubbling in the DOM?

------------------------------------------------------------------------------------------------------------------

It is the hierarchical order in which events propagate in a DOM

Question-5: How do you create, add, and remove elements using JavaScript?

------------------------------------------------------------------------------------------------------------------

a. To create elements we use createElement
b. To add the created element, we use appendChild();. i.e. element's-container.appendChild(newly created element).
c. To remove, we use .remove();
